Mitochondrion Engine

The Mitochondrion Engine is "the power house of the game". It's a very lightweight game engine that I made alone. The most notable feature of it is, voxel rendering. Building a game like Minecraft would be very quick if you used this. But it also supports normal game engine features like particles and normal mesh rendering. Check out the github here
